# Credits

This project is a **rebuild study** — an exercise made to learn, not an original work.

It reconstructs the **aesthetic and interaction** of the portfolio of **Malik Kotb**
(`malik.url`): a scatter of photos floating in 3D on a white background, with size driven by
depth, mouse parallax, drag-to-pan, and hover-to-enlarge.

- **Original design and concept:** Malik Kotb (`malik.url`). All credit for the original idea,
  look, and feel belongs to him.
- **This rebuild:** built independently from scratch (the original source code was not available),
  purely for study. The implementation reproduces the *result*, not Malik's source.
- **Photos:** original content by Andrea Lando (currently placeholders).
- **Typeface:** the chrome uses **PP Neue Montreal Mono** (Pangram Pangram) — the font
  the original site ships — loaded from Malik Kotb's public repo via the jsDelivr CDN,
  with system monospace as the fallback.

No claim of authorship is made over the original design. If you are the original author and want
this taken down or amended, that's entirely your call.
